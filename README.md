# Aumentar-e-Diminuir-Fonte
Aumentar e Diminuir Fonte
Iniciar o sistema
A página é carregada normalmente com um tamanho de letra padrão.
Definir um tamanho de fonte inicial
O sistema guarda um valor inicial para o tamanho da fonte (exemplo: 16px).
 Após isso criar dois botões
Um botão com o texto A+ para aumentar a fonte.
Um botão com o texto A- para diminuir a fonte.
Aguardar a ação do usuário no sistema
O sistema fica esperando o usuário clicar em um dos botões.
Se o usuário clicar no botão A+
Aumentar o valor do tamanho da fonte (exemplo: +2px).
Aplicar o novo tamanho de fonte ao texto da página.
Se o usuário clicar no botão A-
Diminuir o valor do tamanho da fonte (exemplo: -2px).
Aplicar o novo tamanho de fonte ao texto da página.
Limitar o tamanho da fonte
Impedir que a fonte fique pequena ou grande demais (exemplo: mínimo 12px e máximo 24px).
Encerrar o processo
